static:The Element has position according to the normal flow of the document

Relative: "The element is posisioned relative to its normal position. Setting the top, right, bottom, and left properties will cause it to be adjusted away from its normal position..."

Absolute: The element is positioned relative to its closest ancestor (not static). If there are no such ansesstors, it uses the the document body, and moves along with with page scrolling...

Fixed: The element is posisioned relative to the viewport, Which means it always stays in the same place even if the page is scrolled.

Sticky: The element is treated as relative positioned untill it crosses a specific threshold, at which point it is treated as fixed positioned...
